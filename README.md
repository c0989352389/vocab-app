# VocaBoost — 英文單字學習 App

Duolingo 風的單字學習工具,內建 350 字題庫(通用會話 200 + AI 程式 150)。

## 功能
- 🎯 SRS 智能複習(SM-2 簡化版),答錯今日立即再考
- 📚 翻卡 + 聽寫混合學習
- 🔥 連勝、連對 combo、寶石獎勵
- 🔊 Google Translate 發音(連點 4 秒內變慢速)
- 🅰️ 美/英雙音標(dictionaryapi.dev)
- 📊 學習熱力圖 + 熟練度分布圓餅圖
- 💾 JSON 匯出匯入備份
- 📱 PWA — 可裝到手機桌面、離線可用

## 技術
Vite + React 18 + Tailwind + React Router + Zustand + Recharts + vite-plugin-pwa

## 開發
```bash
npm install
npm run dev      # 開發
npm run build    # 打包
npm run preview  # 預覽
```
